记忆不是比特的集合，而是在对称性破缺下幸存的不变循环结构。

# PROBLEM

**痛点定义**: 传统图灵机用比特序列存储记忆，但生物智能系统是非遍历的——它们不会随机游走整个状态空间，而是收敛到低维循环结构。如何在非遍历系统中形式化"记忆持久性"？

**前人困境**:
- 遍历系统依赖测度保持（measure-preservation）：时间平均=系综平均
- 智能系统打破对称性，表现路径依赖，熵最小化（非熵守恒）
- 经典方法无法解释为何某些模式"记住"而其他模式"遗忘"

# INSIGHT

**核心直觉**: 循环闭合 (cycle closure) 是记忆的代数本质。当系统被迫在多个等价选择中挑选一个，对称性破缺产生循环：选择的结果+竞争的备选+它们之间的转换关系被组织成闭合的循环。边界算子 ∂² = 0 确保"边界的边界消失"，过滤掉不一致的碎片，只有拓扑闭合的循环幸存为记忆。

**关键步骤**:
1. **Dot-Cycle 二分法**: 开放链（碎片）在边界算子下崩溃为平凡点（H₀，无关系内容）；闭合循环无边界，幸存为非平凡同调类（H₁，持久记忆）
2. **振荡相位编码 + 符合检测**: 神经振荡将线性时间折叠到圆 S¹ 上，符合窗口内的尖峰对齐触发闭合（∂γ = 0），未对齐的尖峰作为开放边界被剪枝

# DEEP DIVE

## 核心机制解构

### 1. 拓扑视角：从测度守恒到循环保持

**经典遍历系统**:
- 概率测度 μ 在变换 T 下不变：μ(T⁻¹A) = μ(A)
- 长期时间平均 = 系综平均（统计力学基础）

**非遍历认知系统**:
- 不保持测度，保持同调类：[T∗γ] = [γ] ∈ Hₖ(X)
- 轨迹收敛到低维吸引子（不是均匀分布）
- 循环成为信息载体（不是分布）

**引理 1（对称性破缺产生不变循环）**:
当扰动 ε 打破对称群 G 时，残余不变量表现为闭合循环 γ ⊂ Z，在反馈下稳定（∂γ ≅ 0）。

### 2. 信息论视角：熵最小化 vs 结构保持

**引理 4（对称性破缺促进预测）**:
- 对称态：所有结果等价 → 熵 H = log|G| 最大 → 无预测力
- 破缺后：残余循环编码持久结构 → 熵降低 → 预测成为可能

**推论 2（预测 = 测度集中在循环上）**:
非遍历系统能预测，当且仅当概率测度 μₜ 在 t→∞ 时集中在不变循环 [γ] ∈ Hₖ(X) 上。

### 3. 生物实现：多重同步神经群（PNGs）

**定理 2（多重同步闭合 → 带序不变读取的神经 1-循环）**:
给定有向突触图 G = (V,E)，延迟 dᵢⱼ 和 STDP 权重 wᵢⱼ：
- 符合窗口 Δ > 0
- 载流子路径 γ = (v₁ → v₂ → ⋯ → vₘ → v₁)
- 若 |∑ᵐᵢ₌₁ dᵢᵥᵢ₊₁ - nTθ| ≤ Δ 且 ∏ᵐᵢ₌₁ wᵢᵥᵢ₊₁ ≥ τ（循环增益阈值）

则：
1. 重入闭合：泊松卡莱映射在相位环上承认稳定不动点
2. 序不变读取：任何保持符合性的微扰都产生相同闭合循环
3. 定时感知链复形的同调类 [γ] ∈ H₁(Z) 为非平凡

**生物意义**:
- Theta 振荡（4-8 Hz）提供脚手架（Ψ）
- Gamma 爆发（30-100 Hz）编码内容包（Φ）
- STDP 强化闭合路径，抑制未闭合路径

### 4. 层次嵌套：自举记忆巩固

**命题 4（跨频段嵌套诱导循环层次）**:
- Gamma PNGs（微循环）在 theta 相位窗内组装
- Theta 循环（中循环）在睡眠-觉醒周期内通过涟漪重放巩固
- 清醒生成探索性开放轨迹（σ ∈ C₁）

**推论 4（睡眠-觉醒循环作为拓扑闭合）**:

### 5. 高阶不变：意识的层次闭合

**原则 4（Sheaf-Cosheaf 闭合与意识）**:
- Sheaf F（感知）：局部碎片 → 全局粘合
- Cosheaf G（行动）：全局计划 → 局部分解

**定理 5（意识作为不变循环闭合）**:
给定链复形 (C•(Ψ), ∂)，意识对应于：
1. **整合**：∂² = 0 确保局部不一致性消失，事件绑定为全局不变循环
2. **分化**：不同循环 [γ₁], [γ₂], ⋯ ∈ Hₖ(C•) 提供独立内容

## 关键定理证明草图

### 定理 1（循环编码序不变）

**陈述**: 设 (Z, x₀) 为带基点的状态空间，A = {a₁, ⋯, aₙ} 为局部移动。对任意产生循环 γₘ 在 x₀ 的有限移动序列 aᵢ₁ ⋯ aᵢₘ：

首个同调类 [γₘ] ∈ H₁(Z) 仅依赖于：
- 构成移动的**多重集合**（使用计数）
- **不依赖**执行顺序

**证明要点**:
1. Abelian 性质：加法算子的交换性
2. 连接局部移动形成循环：γ = Σᵢ eᵢ（定向边）
3. Hurewicz 同态 h: π₁(Z, x₀) → H₁(Z) abelianizes 路径组合
4. 因此 [γₘ] 仅依赖边的和（模边界），不依赖顺序

### 引理 6（符合诱导闭合）

**陈述**: 设 N = {1, ⋯, n} 为神经元集，产生尖峰序列 S = {(i, tₖ)}，相位 φ(tₖ) ∈ S¹。固定符合窗口 Δ ∈ (0, π) 和符合关系 i ⇄ j 若 |φ(i) - φ(j)| ≤ Δ。

构造定向 1-骨架 G_Δ(S)：顶点 = N，定向边 e = (i → j) 对每个符合对。

则：
1. **符合闭合**：符合检测器 K_Δ = z_Δ(S) := Z₁(G_Δ) 且 ∂z_Δ(S) = 0
2. **可复现循环的幸存**：K_Δ 移除的边恰好是其净贡献出现在 ∂c_Δ(S) 中的边（未对齐尖峰）

**生物意义**:
- 未对齐尖峰 = 开放边界 = 被剪枝
- 对齐尖峰 = 闭合循环 = 强化为记忆

**推论 7（认知的对角非闭合）**:
设认知将数学作为其自身闭合操作的抽象生成。任何捕获认知的形式系统遭遇对角化障碍（Cantor, Gödel, Turing）。
